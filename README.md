# 🔒 Java Password Strength Checker

A sleek, desktop-based GUI application built with Java Swing that evaluates the security of user passwords in real-time. It features a modern, dark-mode aesthetic with custom rounded UI components and dynamic color-coded feedback.

## ✨ Features

* **Modern Dark UI:** Built with a custom color palette (Deep Dark Mode) for a sleek, professional look.
* **Custom Rounded Components:** Utilizes Java `Graphics2D` rendering to create smooth, anti-aliased rounded panels instead of standard sharp corners.
* **Dynamic Visual Feedback:** The application provides instant, color-coded feedback based on password strength:
  * 🔴 **Red:** Poor / Invalid (Under 8 characters)
  * 🟠 **Orange:** Average
  * 🔵 **Blue:** Good
  * 🟢 **Green:** Excellent (Unbreachable)
* **Comprehensive Validation Logic:** Evaluates passwords based on length, uppercase letters, lowercase letters, digits, and special characters.

## 🛠️ Technologies Used

* **Language:** Java (JDK 8 or higher)
* **Framework:** Java Swing & AWT (Abstract Window Toolkit)
* **Concepts:** Object-Oriented Programming (OOP), Event-Driven Programming, Custom UI Painting (`paintComponent`).
